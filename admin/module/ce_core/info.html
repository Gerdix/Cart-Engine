Documentation of CE Core Module (CECM)

Introduction

This is a multipurpose module for Cart Engine. This module contains support for product feature, custom field, and sub product.

Usage

This module is a multipurpose module for Cart Engine (CECM). Therea are various features in this module to enrich the CE experience.

You can use Module Manager or edit the template directly to add this module to your CE installation.

Featured Products

To display list of products using CECM:

<!-- BEGINMODULE ce_core -->
mode = product_list
items = ( random | best | newest | see_also | cat_featured | site_featured | history | list_of_item_id )
out_of_stock = ( 0 | 1 )
limit = number of products to display
display = ( small | big | slideshow | slider | list_small )
tag = ( new | best | promo )
<!-- ENDMODULE ce_core -->

Parameters:

  • items
    • random: display random products
    • best: display best seller products
    • newest: display newest products
    • see_also: display 'see also' products for a specific product (in product detail screen)
    • cat_featured: display featured products for a specific category (in product list screen)
    • site_featured: display site-wide featured products
    • history: display history of user visited products
    • list_of_item_id, eg: 1,2,3,4,5: display specific products, separated with commas.
  • out_of_stock: 0 = hide out of stock products (default), 1 = also display out of stock products
  • limit: 10 = limit number of products to display (default 10)
  • display
    • small: display in small design
    • big: display in bigger design
    • slideshow: display in slideshow design
    • slider: display in slider design
    • list_small: display in vertical list with small thumbnail
  • tag
    • new: display 'new' tag
    • best: display 'hot' tag
    • promo: display 'promo' tag
  • PS: some parameters may not be able to be combined with other parameters, eg: tag parameter can only be used with display small & big.

Examples:

<!-- BEGINMODULE ce_core -->
mode = product_list
items = best
display = big
tag = best
<!-- ENDMODULE ce_core -->

<!-- BEGINMODULE ce_core -->
mode = product_list
items = 1,2,3,4,5
display = slider
<!-- ENDMODULE ce_core -->

Sub Products

You can also use CECM to display list of sub products for a specific product.

<!-- BEGINMODULE ce_core -->
mode = sub_product
item_id = the item id
<!-- ENDMODULE ce_core -->

Examples:

<!-- BEGINMODULE ce_core -->
mode = sub_product
item_id = 123
<!-- ENDMODULE ce_core -->

Custom Fields

You can also use CECM to display list of custom fields for a specific product.

<!-- BEGINMODULE ce_core -->
mode = custom_field
item_id = the item id
<!-- ENDMODULE ce_core -->

Examples:

<!-- BEGINMODULE ce_core -->
mode = custom_field
item_id = 123
<!-- ENDMODULE ce_core -->
